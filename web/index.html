osulyzer

an app that takes in osu!standard data from the osu! api and does interesting things with it



enter your user id, then click either button to see interesting stats!

- rankup! -
tells you how many top plays of a certain pp value you will need to make in order to reach a four digit rank (>#10,000)

- active users stats -
tells you the total number of active users (users that have submitted a play in the last month) and what percent you place.
